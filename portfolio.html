Rock-Paper-Scissors

A game of Rock-Paper-Scissors. It was originally designed to be played in the console, but was later updated to engage the player with buttons and user input. This project was completed as an introduction to Javascript and as a part of the Odin Project.


Etch-a-Sketch

An interactive Etch-a-sketch made using HTML, CSS, and Javascript. The main focus of this project was going deeper with Javascript. Select the grid size you want and begin drawing! Feel free to clear the grid anytime you want and start over!


Calculator

This project utilizes HTML, CSS, and Javascript to create a functioning calculator. This project main focus was to create a visually appealing interface using CSS and become comfortable manipulating arrays.
